# Front Cover Directory

Place your custom front cover PDF here.

**File name:** `cover.pdf`

**Default usage:**
```bash
make merge EXTERNAL_FRONT=external_covers/front/cover.pdf
```

or if using default paths:
```bash
make merge
```

## Guidelines

- **Format:** PDF (single page)
- **Size:** 8.5" × 11" (US Letter) or A4 (210mm × 297mm)
- **Resolution:** 300 DPI recommended
- **Contents:** Project title, authors, institution, date, logos

See `../README.md` for detailed instructions.
