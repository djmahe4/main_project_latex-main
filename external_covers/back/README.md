# Back Cover Directory

Place your custom back cover PDF here.

**File name:** `cover.pdf`

**Default usage:**
```bash
make merge EXTERNAL_BACK=external_covers/back/cover.pdf
```

or if using default paths:
```bash
make merge
```

## Guidelines

- **Format:** PDF (single page)
- **Size:** 8.5" × 11" (US Letter) or A4 (210mm × 297mm)
- **Resolution:** 300 DPI recommended
- **Contents:** Acknowledgements, summary, contact info, QR code, etc.

See `../README.md` for detailed instructions.
